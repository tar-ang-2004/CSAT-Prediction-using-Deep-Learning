About DeepCSAT

AI-powered customer satisfaction prediction using deep learning

Project Overview

DeepCSAT is an advanced machine learning solution designed to predict Customer Satisfaction (CSAT) scores for e-commerce customer support interactions. By leveraging deep learning neural networks, the system analyzes multiple dimensions of customer interactions to provide accurate satisfaction predictions.

In today's competitive e-commerce landscape, understanding customer satisfaction is crucial for business success. This project bridges the gap between raw interaction data and actionable insights, enabling businesses to proactively improve customer experience and reduce churn.

Project Objectives

  • Build accurate deep learning model for CSAT prediction
  • Identify key factors influencing customer satisfaction
  • Provide real-time prediction capabilities
  • Enable data-driven service improvement decisions

Technical Stack

Backend Flask (Python)
Frontend Tailwind CSS
ML Framework TensorFlow/Keras
Data Processing Pandas, NumPy
Visualization Plotly, Seaborn

Input Layer

Accepts 21 engineered features including sentiment scores, response times, agent metrics, and product information.

Hidden Layers

Multiple dense layers with ReLU activation, batch normalization, and dropout for regularization to prevent overfitting.

Output Layer

Single neuron with linear activation to predict continuous CSAT score on a scale of 1 to 5.

Input Features

Customer Interaction

  • Response Time (hours)
  • Connected Handling Time
  • Issue Frequency
  • Channel Name (encoded)
  • Issue Hour & Day

Sentiment Analysis

  • Positive Sentiment Score
  • Negative Sentiment Score
  • Remarks Length
  • Has Remarks Flag

Agent Performance

  • Agent Case Count
  • Tenure Bucket (encoded)
  • Agent Shift (encoded)

Product & Context

  • Item Price
  • Product Category (encoded)
  • Price Category (encoded)
  • Customer City (freq encoded)
  • Sub-category (freq encoded)

Dataset Information

Overview

The model is trained on a comprehensive e-commerce customer support dataset containing approximately 85,000 real customer interactions across various channels, product categories, and support scenarios.

Total Records ~85,000
Features 21 Engineered
Target Variable CSAT Score (1-5)

Data Sources

  • Customer support tickets and interactions
  • Agent performance metrics and profiles
  • Product and order information
  • Customer satisfaction survey responses
